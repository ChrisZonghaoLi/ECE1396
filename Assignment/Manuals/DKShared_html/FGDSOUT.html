DESCRIPTION

The product FGDSOUT is a specific sub product used for tiling .

It provides procedures for the Merge of layout files (gds2 description format.

It provides Hierachical Cells Management .

REQUIREMENTS

The scripts developed in FGDSOUT require a Tcl/Tk (included in Design KIT) and MIF (rel.>=1.7.5) (included in Design Kit).
