ASSOCIATED KEY

calibreSTSwitchesHelp

DESCRIPTION

calibreSTSwitchesHelp calibreSTSwitchesHelp extract Documentation about switches from calibre cgi file : either drc antennadrc lvs tiling or drc lvs tiling drcMask lvsMask tilingMask To Get Help on drc switches, type Unix command : calibreSTSwitchesHelp -drc To Get Help on lvs switches, type Unix command : calibreSTSwitchesHelp -lvs To Get Help on tiling switches, type Unix command : calibreSTSwitchesHelp -tiling .... To Get Help on calibreSTSwitchesHelp command itself, type Unix command : calibreSTSwitchesHelp -help

Options

[-drc]
ST Drc switches doc. Type: 'bool', optional, default: 'false'
[-ant]
ST Antenna switches doc. Type: 'bool', optional, default: 'false'
[-lvs]
ST Lvs switches doc . Type: 'bool', optional, default: 'false'
[-tiling]
ST Tiling switches doc. Type: 'bool', optional, default: 'false'
[-verbose]
Verbose execution. Type: 'bool', optional, default: 'false'
[-debug]
Debug execution. Type: 'bool', optional, default: 'false'
[-arguments]
Display the script parameter description. Type: 'bool', optional, default: 'false'
[-help|-h|-u|-U]
Display the script usage. Type: 'bool', optional, default: 'false'
[-gui]
Graphical user interface. Type: 'bool', optional, default: 'false'

Limits

  • must choose either drc or ant or lvs or tiling
  • must choose either drc or ant but not both
  • must choose either drc or lvs but not both
  • must choose either drc or til but not both
  • must choose either ant or lvs but not both
  • must choose either ant or til but not both
  • must choose either lvs or til but not both
