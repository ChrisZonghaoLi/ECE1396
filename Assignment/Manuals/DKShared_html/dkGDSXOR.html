dkGDSXOR

Design-Kit XOR between two Gds files with Calibre Procedure

ASSOCIATED KEY

dkGDSXOR

DESCRIPTION

dkGDSXOR compare gds2 file. It does not require any Design-Kit to run but Calibre ctrl file and gdsfilter setup file must be specified. Specifying the output gds2 file name is mandatory. It is optional to specify the original gds2 File Name and cell name. This all ows to update the ctrl file and the setup file for an other cell and generate dummies for that cell.
