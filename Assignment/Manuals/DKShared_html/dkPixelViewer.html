SYNOPSIS

[-legend <legend File>]

[-xscreen <XScreenSize >]

[-yscreen <YScreenSize >]

[-xgraph <xgraph size >]

[-ygraph <ygraph size >]

[-verbose]

[-debug]

[-arguments]

[-help|-h|-u|-U]

[-gui]

DESCRIPTION

dkPixelViewer Display Pixels on various colours. Pixels are read in a input file. This input file may be generated with calibre tool.

PURPOSE
